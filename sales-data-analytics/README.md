
# Sales Data Analytics Project

This project demonstrates end-to-end Data Analytics workflow using Python, SQL, and Power BI.

## Project Structure

- data/cleaned_data.csv
- database/sales.db
- sql/analysis_queries.sql
- powerbi/sales_dashboard.pbix

## Steps

1. Data Preparation (Python & Pandas)
2. Database Creation (SQLite)
3. SQL Analysis
4. Power BI Dashboard
5. Professional formatting, titles, sorting, and theme.

Skills: Python, Pandas, SQLite, SQL, Power BI, Data Visualization, Dashboard Design
