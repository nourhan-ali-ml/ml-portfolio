# Customer Segmentation Using K-Means and PCA

This project demonstrates customer segmentation using unsupervised learning techniques to identify patterns in customer behavior.

## Project Overview
- **Dataset:** E-commerce Customer Behaviour Dataset
- **Objective:** Segment customers into meaningful groups
- **Techniques:** K-Means Clustering, PCA
- **Tools & Libraries:** Python, pandas, numpy, scikit-learn, matplotlib, seaborn
- **Environment:** Google Colab

## Key Steps
- Data preprocessing and scaling
- Feature selection
- Dimensionality reduction using PCA
- Customer clustering with K-Means
- Visualization of clusters

## Outcome
The project provides insights into customer behavior that can support marketing, personalization, and business strategies.

## Skills Demonstrated
- Unsupervised learning
- Clustering techniques
- Dimensionality reduction
- Business data analytics
